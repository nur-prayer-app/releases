Install Guide

Get Nur on any device

Nur works everywhere — as a downloadable Windows app or an installable web app on your phone, tablet, or desktop browser. Pick your platform below.

iPhone & iPad

Safari only
1

Open nur-prayer-app.github.io/app in Safari

2

Tap the Share button at the bottom of the screen

3

Scroll down and tap "Add to Home Screen"

4

Tap "Add" in the top right. Nur appears on your home screen as a full-screen app.

Important: This only works in Safari. Chrome, Brave, and Firefox on iOS cannot install web apps. If you're using another browser, open the link in Safari first.

Android

Chrome / Brave / Edge
1

Open nur-prayer-app.github.io/app in Chrome, Brave, or Edge

2

You should see an "Install Nur" banner at the bottom. Tap Install.

3

If no banner appears: tap the three-dot menu (top right) and select "Add to Home screen" or "Install app"

Tip: Once installed, Nur runs full-screen with no browser chrome — just like a native app. It works offline too.

Windows

Desktop app
1

Download the installer (one file, ~82 MB)

2

Run the downloaded file and follow the setup wizard

3

Nur installs to Program Files with a desktop shortcut and system tray icon

Note: Windows SmartScreen may show a warning on first run (the app isn't code-signed yet). Click "More info" then "Run anyway".

Mac, Linux & Desktop Browsers

Chrome / Edge
1

Open nur-prayer-app.github.io/app in Chrome or Edge

2

Click the install icon in the address bar (a monitor with a down arrow), or use the three-dot menu → "Install Nur"

3

Nur opens as a standalone window — no tabs, no address bar. Find it in your app launcher.
